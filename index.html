Linguagem de Programação Java

Definição Características Sintaxe POO Tipos de Dados

Definição:

Java é uma linguagem de programação de alto nível e uma plataforma de computação amplamente utilizada, criada pela Sun Microsystems em 1995. Ela é orientada a objetos, o que significa que segue um paradigma que organiza o código em classes e objetos, facilitando a reutilização e manutenção. A principal característica do Java é sua independência de plataforma, permitindo que programas sejam escritos uma vez e executados em qualquer lugar, graças à Java Virtual Machine (JVM).

Características :

Java é conhecido por sua robustez, segurança e versatilidade. Ele utiliza o conceito de bytecode (é uma fase intermediária do código entre o código-fonte, escrito numa linguagem de programação específica, e a aplicação final), que é interpretado pela JVM (Java Virtual Machine),o que garante que o mesmo código funcione em diferentes sistemas operacionais, como Windows, Linux e macOS. Além disso, o Java é amplamente utilizado em aplicações corporativas, desenvolvimento de aplicativos Android, sistemas embarcados e até mesmo em servidores web.

Sintaxe:

A Sintaxe Java é o conjunto de regras e comandos obrigatórios para escrever programas que o computador entende, funcionando como a gramática da linguagem. Para o código rodar, é preciso seguir essa escrita de forma rígida, usando chaves para abrir e fechar blocos de código e ponto e vírgula para encerrar cada instrução. Além disso, o Java diferencia letras maiúsculas de minúsculas, exigindo muita atenção ao digitar.

POO:

A sintaxe do Java foi desenhada para seguir a Programação Orientada a Objetos (POO) , um modelo que organiza o código tentando imitar o mundo real. Em vez de uma lista gigante de comandos, o programa é dividido em partes menores que se baseiam em coisas reais. Toda a estrutura do sistema funciona como um ecossistema onde vários desses objetos conversam entre si para realizar as tarefas.

Essa organização funciona através de Classes e Objetos. A classe é o molde abstrato, como um cortador de biscoitos ou a ideia geral de um "Programador". Já o objeto é o resultado real feito a partir do molde, como o biscoito pronto ou um profissional real como o "Programador João". Para a simulação ser completa, cada objeto junta em si mesmo suas características próprias e suas ações práticas.

Tipos de Dados:

O Java é uma linguagem fortemente tipada, o que significa que toda variável precisa ter um tipo definido para que o computador saiba como processar e armazenar o dado. Esses tipos são divididos em dois grandes grupos: os tipos primitivos, que são os dados mais básicos e nativos da linguagem usados para guardar valores simples diretamente na memória (como números inteiros, números decimais, caracteres únicos e valores lógicos de verdadeiro ou falso); e os tipos de referência, que são estruturas mais complexas que guardam o endereço de memória onde a informação real está guardada, como textos completos e listas de dados.

Dentro do grupo dos tipos de referência, existem também os tipos definidos pelo usuário. Eles surgem quando o programador cria suas próprias classes no código, transformando-as em tipos de dados personalizados. Isso permite agrupar diferentes tipos primitivos, textos e regras específicas dentro de uma única estrutura customizada para atender às necessidades do programa.
